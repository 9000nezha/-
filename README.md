# 枪炮内弹道学数值解法程序

## 程序概述

本程序基于《枪炮内弹道学》第2章第2.4节的内容实现，使用**四阶龙格-库塔法**求解量纲为1的内弹道方程组，能够计算并定位以下特殊点：

- **最大压力点**：膛内压力达到最大值的位置
- **燃烧分裂点**：火药燃烧分裂时的位置（Z = 1）
- **燃烧结束点**：火药燃尽时的位置（Z = Zk）
- **炮口点**：弹丸到达炮口时的位置

## 数学模型

### 1. 量纲为1的内弹道方程组

程序求解以下微分方程组（公式2-72）：

```
dZ/dt̄ = √(θ/2B) · p̄^n    (Z < Zk)
dl̄/dt̄ = v̄
dv̄/dt̄ = (θ/2) · p̄
dp̄/dt̄ = [l₀/((l̄+l̄_ψ)v̄)] · [1+Δ(α-1/ρ_p)·p̄]·(dψ/dt̄) - [(1+θ)/(l̄+l̄_ψ)]·p̄·v̄
```

其中：
- Z：相对燃烧厚度
- l̄：量纲为1的行程
- v̄：量纲为1的速度
- p̄：量纲为1的压力
- t̄：量纲为1的时间

### 2. 四阶龙格-库塔法

采用经典的四阶龙格-库塔公式（公式2-73）：

```
y_{i,k+1} = y_{i,k} + (h/6)(K_{i1} + 2K_{i2} + 2K_{i3} + K_{i4})
```

其中K值按以下方式计算：
```
K_{i1} = f_i(x_k, y_{1k}, ..., y_{nk})
K_{i2} = f_i(x_k + h/2, y_{1k} + hK_{11}/2, ...)
K_{i3} = f_i(x_k + h/2, y_{1k} + hK_{12}/2, ...)
K_{i4} = f_i(x_k + h, y_{1k} + hK_{13}, ...)
```

### 3. 黄金分割法求最大压力点

当检测到压力从上升转为下降时，程序使用黄金分割法（比例0.618）精确确定最大压力点的位置。

## 计算公式

### 常量计算

```
phi = phi1 + lambda2 * omega / m
Delta = omega / V0
l0 = V0 / S
vj = sqrt(2 * f * omega / (theta * phi * m))
B = S² * e1² / (f * omega * phi * u1²) * (f * Delta)^(2*(1-n))
lg_bar = lg / l0
```

### 初值计算

```
t0_bar = 0
l0_bar = 0
v0_bar = 0
p0_bar = p0 / (f * Delta)
psi0 = (1/Delta - 1/rho_p) / (f/p0 + (alpha - 1/rho_p))
Z0 = (sqrt(1 + 4*lambda*psi0/chi) - 1) / (2*lambda)
```

### 量纲转换

```
l = l_bar * l0
t = t_bar * l0 / vj
v = v_bar * vj
p = p_bar * f * Delta
```

## 注意事项

1. **步长选择**：
   - 步长h越小，计算精度越高，但计算时间也越长
   - 建议取值范围：0.001 ~ 0.01
   - 调试时可先用较大步长，确认程序正确后再减小步长

2. **精度控制**：
   - 程序使用EPSILON = 1e-6作为基本精度阈值
   - 黄金分割法迭代次数上限为100次
   - 主循环迭代次数上限为100000步

3. **数值稳定性**：
   - 当压力或速度接近零时，可能需要特殊处理
   - 程序会自动检查分母是否接近零

4. **火药形状参数**：
   - Zk（火药燃尽时相对燃烧厚度）默认为1.5
   - 可根据实际火药形状调整此值

## 故障排除

### 程序无法读取输入文件
- 检查文件路径是否正确
- 确认文件格式为纯文本
- 检查参数数量是否正确

### 计算结果异常
- 检查输入参数单位是否正确
- 确认所有参数为正值
- 尝试减小步长

### 程序运行时间过长
- 增大步长h
- 检查是否陷入无限循环（观察输出信息）
